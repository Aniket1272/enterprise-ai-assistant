# Enterprise AI Knowledge Assistant

A production-style Retrieval-Augmented Generation (RAG) backend service built with FastAPI, LangChain, ChromaDB, and OpenAI.

## Tech Stack

- Python
- FastAPI
- LangChain
- ChromaDB
- OpenAI
- Docker

## Features

- Upload PDF documents
- Generate embeddings
- Vector search
- Retrieval-Augmented Generation
- REST APIs
